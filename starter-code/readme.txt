Cluedo game:

To initialize the game type cluedo."the action you want to take", here are some rules:

- You can draw as many cards as you want with cluedo.drawCards()
- The first action in the game will be tipying cluedo.drawBaddie() so a villain is initialized
- To take a guess in who is the villain you have to write the function like this:
	. cluedo.guess("Person", "Room", "weapon") <-- The first two with capital letters.
- Once you guess the villain you should recieve a "You got him".
- You have 5 tries to guess the villain.
